# Krishi Saathi 🌾

Krishi Saathi is an AI-powered platform designed to help farmers access government schemes, market information, and modern agricultural resources.

## Features

- AI assistant for scheme eligibility
- Voice interaction
- Accessibility controls
- Market advisory
- Scam alert detection
- Confidence score
- Nearby help centers
- Community insights
- Government training programs
- Subsidized farming equipment
- Weather-based farming advice
- Agri-technology tools

## Architecture

Frontend: HTML / CSS / JavaScript  
Hosting: AWS S3  
AI Processing: Amazon Bedrock  
Backend Logic: AWS Lambda  
Database: DynamoDB
